# chrome-extensions

## Current list
- Download Organizer
> This extension should run in the background and organizes files downloaded based on their source websites.
- MODIFIED IMDownloader
> This extension is a reverse engineered and updated version of [IMDownloader](https://chromewebstore.google.com/detail/imdownloader-download-sto/fkilgeompmipodhkdmafejpfgnfbjmkk?hl=en-US&utm_source=ext_sidebar).
> It was updated as the current version (3.1.9 as of Feb 8 2026) appears to skip some of the files being downloaded

## Quick Instructions
1. Go to chrome://extensions/
2. Enable Developer mode (top right switch)
3. Click 'Load unpacked'
4. Choose the folder where the files (manifest.json, background.js, etc) is stored
